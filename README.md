# Analyse exploratoire pour le projet d’expansion internationale d'Academy

## Contexte

Dans cet exercice, vous êtes **Data Scientist** dans une start-up EdTech nommée **Academy**, qui propose des contenus de formation en ligne pour un public de niveau lycée et université.  

Mark, votre manager, vous a présenté le projet d’expansion à l’international de l’entreprise et vous confie une première mission d’analyse exploratoire. L’objectif est de déterminer si les données sur l’éducation de la Banque mondiale peuvent enrichir la réflexion autour du projet d’expansion.

Vous avez déjà réduit le périmètre des données en utilisant votre compréhension métier et en évaluant la qualité des jeux de données. Vous pouvez maintenant vous concentrer sur **l’analyse des indicateurs sélectionnés** pour répondre à la demande de Mark.

---

## Objectifs

- Identifier les pays avec un **fort potentiel de clients** pour les services de Academy.  
- Évaluer **l’évolution possible de ce potentiel**.  
- Réaliser des **analyses statistiques univariées et bivariées**.  
- Formaliser le travail pour qu’il soit **réutilisable par vos collègues**.  

---

## Étapes du projet

1. **Exploration initiale des fichiers de données**
   - Identifier le fichier principal (`Data`) parmi les différents fichiers disponibles.  
   - Comprendre la structure et le contenu des données.  

2. **Nettoyage et préparation des données**
   - Détecter et traiter les valeurs manquantes.  
   - Identifier et supprimer les doublons.  
   - Supprimer les colonnes inutilisables ou complètement vides.  

3. **Réduction du périmètre des indicateurs**
   - Identifier un nombre d’indicateurs pertinent pour l’analyse.  
   - Faciliter le choix des indicateurs afin d’orienter la prise de décision.  

4. **Analyse exploratoire**
   - Statistiques univariées : résumé des colonnes numériques et catégorielles.  
   - Statistiques bivariées : identifier des relations entre indicateurs et pays.  
   - Visualisations pour faciliter la compréhension des données.  

5. **Synthèse et recommandations**
   - Déterminer les pays les plus prometteurs pour l’expansion.  
   - Formaliser les résultats pour qu’ils soient exploitables par l’équipe.  

---

## Observations initiales

- Le fichier **Data** est le plus important et contient le plus grand nombre d’indicateurs.  
- Le nombre d’indicateurs est **trop élevé pour une analyse manuelle complète**.  
- Il est donc nécessaire de **réduire le périmètre des données** pour se concentrer sur les indicateurs les plus pertinents pour la demande de Mark.  

---

## Résultat attendu

À la fin de l’exercice, vous aurez :  

- Une sélection d’indicateurs pertinents pour l’analyse.  
- Une exploration claire des pays présentant un fort potentiel pour Academy.  
- Des visualisations et un rapport synthétique permettant de **prendre des décisions éclairées** pour l’implantation internationale.

---

## Technologies utilisées

- Python

- Matplotlib / Seaborn pour les visualisations
